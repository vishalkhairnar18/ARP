# Bhairavnath Construction - Portfolio Website

## Overview
A complete PHP + MySQL dynamic portfolio website for a Civil Engineer/Builder with Admin Panel (ARP).

**Note**: This project is designed to run on **local XAMPP** environment, not Replit.

## Project Structure
- `/admin/` - Admin panel files (dashboard, project/service management, inquiries, profile)
- `/include/` - Config and shared components (header, footer)
- `/assets/` - CSS, JS, and images
- `/uploads/` - User uploaded files (projects, profiles, services)
- Main pages: index.php, about.php, projects.php, services.php, contact.php

## Database
- **Database Name**: bhairavnath_construction
- **SQL File**: DATABASE.sql (import into phpMyAdmin)
